Features & architecture of LambdaKG

we detail two major types of text- based KG representation (discrimination-based and generation-based) and the proposed prompt learn- ing method for KG representation learning, which are all integrated into LambdaKG.

Model Hubs

LambdaKG consists of a Model Hub which supports many representative text-based KG representation models.

KGT5

A sequence-to-sequence T5-small model from scratch - not initialize with the pre-trained LM weights. The task the model is trained on is head/tail prediction.

SimKGC

SimKGC adopts a biencoder architecture. Two encoders are initialized with the same pre-trained language model but do not share parameters

GPT-3

GPT-3 (Generative Pre-trained Transformer 3) is a natural language processing model developed by OpenAI that uses large-scale neural networks to generate natural language text.

KNN-KGE

We propose kNN-KGE, a new knowledge graph embedding approach with pre-trained language models, by linearly interpolating its entity distribution with k-nearest neighbors.

Support LLM based on API

We provide varous prompt for knowledge probing with LLM with OpenAI's API.
